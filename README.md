# Tour & Tourism Website: TripWithEase

## Project Description
This is a comprehensive tour and tourism web-based platform that allows users to explore various travel destinations, browse available accommodations, check weather, and make bookings. The platform provides an immersive experience with high-quality images, featured sections, and modern design to enhance user experience.

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YASH2204V/trip-with-ease.git
   cd your-project
   ```

2. **Initialize the project**:
   ```bash
   npm init
   ```

3. **Install the dependencies**:
   ```bash
   npm install
   ```

4. **Set up environment variables**:
   - Create a `.env` file in the root of your project.
   - Add the following tokens for Google Authentication:
     ```
     GOOGLE_CLIENT_ID=your-google-client-id
     GOOGLE_CLIENT_SECRET=your-google-client-secret
     EXPRESS_SESSION_SECRET=your-session-secret
     ```

5. **Start the application**:
   ```bash
   npm start
   ```

6. **Access the application**:
   Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

## Features

- **Weather Display**: Users can view real-time weather conditions for different destinations.
- **Location View**: Displays the geographic location of tourist spots, possibly integrated with maps.
- **Calendar with Time**: A built-in calendar to select dates and times for travel or bookings.
- **Specific Images**: Showcases high-quality images of destinations to give users a visual feel of the locations.
- **Featured Sections**:
  - **Arctic**: Stunning destinations in the Arctic region.
  - **Iconic**: Popular and iconic tourist attractions.
  - **Cabins**: Cozy and scenic cabin getaways.
  - **Lakes**: Beautiful lakeside destinations.
  - **Mansions**: Luxury mansion experiences.
- **Modern Design**: A sleek, contemporary design for an enhanced user experience.
- **Animated Footer**: Stylish, animated footer to provide a dynamic finish to the user interface.

## Usage

This is a tour and tourism website with several features to explore:

- **Categories**: Browse through various sections like houses and hotels, each categorized under themes such as Arctic, Mansions, Lakespots, Boats, Amazing, and more.
- **Hotel Reservation**: Check out available hotels in the category of your choice and make reservations.
- **Weather Check**: View the current weather conditions for the location you're interested in visiting.
- **Calendar and Location**: Plan your trips with the integrated calendar, and check the location of the hotel or destination.
- **Hotel Room Views**: See hotel rooms from 5 different angles to get a comprehensive view before booking.
- **User Reviews and Ratings**: Comment on your experience, give star ratings, and check out reviews from other travelers.

## Technologies Used

- **EJS (Embedded JavaScript Templating)**: Used for rendering dynamic HTML pages on the server side.
- **Tailwind CSS**: A utility-first CSS framework to style the application quickly and efficiently.
- **Express**: A minimal and flexible Node.js web application framework, providing a robust set of features for building web and mobile applications.
- **Google Auth**: Integrated for secure user authentication via Google accounts.
- **Node.js**: Used as the runtime environment for executing JavaScript on the server side.
